notify-send

语法

```shell
notify-send [options] {summary} [body]
```

options:

+ -?, --help
+ -a，--app-name
+ -A, --action=[name=]Text
+ -u, --urgency=LEVEL
+ -t, --expire-time=TIME
+ -i, --icon=ICON
+ -c, --category=TYPE[,TYPE…]
+ -h, --hint=TYPE:NAME:VALUE
+ -p, --print-id
+ -r --replace-id=REPLACE_ID
+ -w, --wait
+ -e, --transient
